
📘 Bluetooth Controlled Robot

A mobile-controlled robotic system that uses Bluetooth communication to receive commands from a smartphone and control the movement of the robot in real time.
The system is built using an Arduino microcontroller and a Bluetooth module, enabling wireless control without requiring an internet connection.

⚙️ How It Works

The robot receives commands (forward, backward, left, right, stop) from a mobile application via a Bluetooth module (HC-05/HC-06). These commands are processed by the Arduino, which controls the motor driver to move the robot accordingly.

🔹 Features

Wireless control using Bluetooth

Real-time robot movement

Simple and low-cost implementation

No internet required (offline control)

Easy integration with Android apps

🧩 Components Used

Arduino Uno 

HC-05 Bluetooth Module

L293D Motor Driver

DC Motors

Power Supply (Battery)
